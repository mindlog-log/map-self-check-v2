# MAP Self Check 開発統合メモ

## 画面構成

1. タイトル / 設定画面
   - Mind Logロゴ
   - MAP Self Check
   - デザイン選択
   - 3枚 / 9枚観測選択
   - 観測画面へ進む
   - 利用回数

2. 観測準備画面
   - 選択済みデザインと観測モード
   - シャッフルボタン
   - シャッフル回数
   - 観測結果を開く

3. 観測結果画面
   - 3枚: 横並び
   - 9枚: 3列 × 3段
   - Card番号 / 属性 / 状態コード / テキスト

4. 説明書画面
   - MAP Self Checkの使い方
   - 5属性の役割
   - 状態コード
   - 使い方・注意事項

## シャッフル仕様

- Fisher-Yates方式
- 乱数は `crypto.getRandomValues()`
- シャッフル時にカード順と状態コードを確定
- 観測ボタンではランダム処理しない
- 1回以上シャッフルしないと観測結果を開けない
- シャッフル時にボタンが一瞬発光

## 状態コード仕様

- A: ACTIVE -> `uprightText` を表示
- D/B/O/I/U/S/C -> `abnormalText` を表示
- 画面上に「上」「下」は表示しない
