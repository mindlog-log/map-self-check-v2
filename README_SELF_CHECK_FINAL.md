# MAP Self Check

確認済み仕様:
- 管理者パスワード: LoveLog
- 9枚観測は通常ロック、管理者解除後のみ使用可能
- シャッフルは Fisher-Yates + crypto.getRandomValues()
- シャッフルボタンで山札と状態を確定
- 観測ボタンではシャッフルしない
- 1回以上シャッフルしないと観測結果を開けない
- 78枚テキストはアップロードされたデータをそのまま使用
- Aは上テキスト、D/B/O/I/U/S/Cは下テキストを表示
- 属性画像は M/D/E/T/R にリンク


追加修正:
- 3枚観測の結果カードは「カード番号＋属性｜状態コード」を上部に横書き表示
- 3枚観測の属性画像は薄すぎないように濃く・大きく表示
- 3枚観測は結果文の可読性を最優先
- 9枚観測は画像を非表示にし、番号・属性｜状態コード・結果文の文字配置で見せる
- 9枚観測は文字と余白のバランスを重視
