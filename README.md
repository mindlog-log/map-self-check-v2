# MAP Self Check app starter

MAP自己分析モデル用のReact/Viteスターターです。

## 入っているもの

- 78枚カードデータ: `src/data/cards.js`
- 属性・状態コード定義: `src/data/meta.js`
- 説明書ページデータ: `src/data/guide.js`
- Fisher-Yates + `crypto.getRandomValues()` シャッフル: `src/utils/shuffle.js`
- 3枚観測 / 9枚観測ロック / シャッフル回数 / 利用回数カウンター
- 5属性画像: `src/assets/`

## 重要仕様

- シャッフルボタンを押すたびに78枚全体を1枚単位でシャッフルします。
- 観測ボタンではシャッフルしません。
- 観測ボタンは最後に保存された山札の上から、3枚または9枚を展開します。
- 状態はシャッフル時に確定します。
- 画面上では「上/下」と表示せず、状態コードを表示します。
- `A` の時は元データの「上」の文言、状態異常コードの時は元データの「下」の文言を表示します。

## 起動方法

```bash
npm install
npm run dev
```

## 管理者モード

9枚観測はロックされています。開発用の仮パスワードは `src/App.jsx` の `ADMIN_PASSWORD` を変更してください。
